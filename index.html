Privacy Policy - ALBA SERVICE

Aplikasi ALBA SERVICE dikembangkan untuk membantu pengelolaan nota servis. Kami sangat menghargai privasi Anda. Semua data transaksi disimpan secara LOKAL di perangkat Anda.

  • Data Pelanggan: Nama dan nomor telepon pelanggan disimpan hanya di memori HP Anda untuk keperluan pembuatan nota. Kami tidak mengirimkan data ini ke server pihak ketiga.
  • Izin Bluetooth: Digunakan hanya untuk terhubung dan mencetak nota ke printer thermal Bluetooth milik Anda.
  • Google Drive: Digunakan secara pribadi oleh Anda melalui fitur Backup. File cadangan akan disimpan di akun Google Drive pribadi Anda, dan kami tidak memiliki akses ke file lain di Drive Anda.

Data Deletion Instructions (Instruksi Penghapusan Data)

Sesuai dengan kebijakan Google Play, kami memberikan hak penuh kepada pengguna untuk menghapus data mereka kapan saja karena data bersifat lokal:

  1. Menghapus Nota Tertentu: Buka aplikasi, masuk ke menu Riwayat, pilih nota yang ingin dihapus, lalu klik ikon Hapus.
  2. Menghapus Seluruh Data Aplikasi: Anda dapat menghapus seluruh database secara permanen melalui Pengaturan HP > Aplikasi > ALBA SERVICE > Penyimpanan > Hapus Data (Clear Data).
  3. Menghapus Cadangan Google Drive: Jika Anda pernah melakukan backup, Anda dapat menghapusnya secara manual dengan membuka aplikasi Google Drive dan menghapus file cadangan aplikasi di sana.

Setelah langkah di atas dilakukan, data Anda akan terhapus sepenuhnya dan tidak dapat dipulihkan.
